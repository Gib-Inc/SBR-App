Load that product’s BOM: each component item id + requiredPerUnit.

For each component in the BOM:

If validation passes for all components, apply transactions in one operation:

For the finished product at Hildale:

Create an InventoryTransaction of type PRODUCE with itemType = "FINISHED", location = "HILDALE", quantity = produceQty.

Increase its hildaleQty by produceQty.

For each BOM component (raw item):

Create an InventoryTransaction of type PRODUCE with itemType = "RAW", appropriate location (e.g. "HILDALE"), and quantity = totalRequired.

Decrease that raw item’s stock by totalRequired.

All quantity updates must go through the existing transaction system (no direct field edits).
